# 👨‍💻 Precious Musole — Portfolio Website

This is my personal developer portfolio website showcasing my skills, projects, experience, and technical journey as a **Software Engineer and Computer Science student from Malawi**.

🌐 Live Site: https://precious-musole-web.vercel.app/

---

## 🚀 About The Project

This portfolio is a fully responsive, modern developer website built to present who I am, what I build, and what I’m currently learning.

It dynamically loads content from a `data.json` file, making it easy to update without touching core code.

---

## ✨ Features

- ⚡ Fast static site (GitHub Pages hosted)
- 🎨 Modern UI with terminal-inspired design
- 📱 Fully responsive (mobile + desktop)
- 🧠 Dynamic rendering using JavaScript (`data.json` driven)
- 🌙 Light/Dark theme toggle
- 🧑‍💻 Sections:
  - Hero introduction
  - About me
  - Skills
  - Experience timeline
  - Projects showcase
  - Articles/blog section
  - Contact section
- 🔍 SEO optimized (meta tags + Open Graph + JSON-LD schema)
- 📈 Google indexing ready

## 📦 How It Works

- All content is stored in `data.json`
- `main.js` reads the file and renders sections dynamically
- No need to edit HTML for updates (only JSON)

---

## 🌐 Deployment

Hosted using **GitHub Pages**

To deploy:
1. Push to GitHub repository
2. Go to Settings → Pages
3. Select branch `main`
4. Site goes live automatically
